Java Script

JavaScript é uma linguagem de programação essencial para tornar páginas da web interativas. É usado tanto no front-end (navegador) quanto no back-end (com Node.js), com aplicações em animações, jogos e mais. É flexível, moderno e amplamente utilizado


C#

C# é uma linguagem de programação criada pela Microsoft, amplamente usada para desenvolver aplicativos, jogos (especialmente com Unity), e muito mais. É moderna, robusta e oferece segurança com sua tipagem forte. Ideal para desenvolvimento profissional.


Linguagem R

R é uma linguagem de programação focada em estatística, análise de dados e criação de gráficos. Muito usada em ciência de dados, ela possui uma ampla coleção de pacotes para diversas aplicações. Excelente para análise e visualização de dados
